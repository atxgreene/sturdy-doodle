**3/** The substrate is called Mnemosyne. `pip install

**4/** Memory: 6 tiers (instinct → hot → warm → cold → pattern →
identity). SQLite + FTS5 with ACT-R decay + Hebbian reinforcement.

50K rows: search 2.4 ms, decay 6.15 s. No vector DB.

**5/** Identity: stable across 19 LLM backends.

GPT-4 → back. Identity holds. Tested against a 40-prompt jailbreak
suite. Slip rate measured.

Most agents lose persona on model swap. This one doesn't.

**6/** Self-regulation: SVG dashboard with 17 derived agent traits.

Avatar evolves in real time. Every visual change maps to one
telemetry event you can grep. No black-box "personality engine."

mnemosyne-serve & ; open http://127.0.0.1:8484/ui

**7/** Self-calibration (v0.6): predictions are first-class telemetry.

The agent emits a confidence number. The system observes the outcome.
Calibration becomes a measurable agent trait. Triage clusters
overconfident-wrong claims separately from low-confidence drift.

**8/** Self-improvement (v0.2 onward): triage → proposer → apply,
running autonomously inside the daemon.

The agent observes its own failures, drafts proposals, applies them
with measurement. CREAO-inspired closed loop. No external orchestrator.

**9/** v0.9 gave Instinct its own tier (L0). User-pattern signals
distilled offline into a fast-path system block.

Tell the agent 3× you prefer terse responses. Next session it's

Live-model upper bound is yours to measure. Run on your local box:

**11/** Cognitive-OS checklist (docs/COGNITIVE_OS.md): 5 properties,

Marketing language gated by tests.
